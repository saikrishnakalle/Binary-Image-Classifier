# Binary Image Classifier

A deep learning-based Flask web application that automatically classifies uploaded images as:

- Photograph
- Signature

The system can:

- Detect incorrect uploads
- Detect two photos
- Detect two signatures
- Automatically swap photo and signature if uploaded in wrong positions

## Technologies Used

- Python
- Flask
- TensorFlow
- Keras
- OpenCV
- HTML
- CSS

## Installation

pip install -r requirements.txt

## Run

python app.py

Open:

http://127.0.0.1:5000
